3. Dekripsi Program
 - Project 1: Sistem Parkir Mobil — Mall Prienal
Program manajemen parkir berbasis console yang memungkinkan petugas mencatat kendaraan masuk, menampilkan daftar mobil yang sedang terparkir beserta durasi parkirnya, dan melakukan pembayaran berdasarkan durasi. Biaya dihitung per menit (Rp2.000/menit). Kapasitas maksimal 20 kendaraan, dan setiap data kendaraan dialokasikan secara dinamis menggunakan pointer.

 - Project 2 : Game Tebak Kata
Dua file ini membentuk game tebak kata dengan fitur tebakan per huruf, sistem nyawa, hint, perhitungan skor berdasarkan sisa nyawa dan tingkat kesulitan, serta leaderboard 5 besar. Header (TEBAKKATA_H) mendefinisikan struktur data KataGame dan Skor, serta deklarasi semua fungsi yang diimplementasikan di file terpisah..

 - Project 3 : Sistem Manajemen Inventaris Toko
Program pengelolaan stok barang toko yang mendukung penambahan barang baru (dengan validasi kode unik), tampilan seluruh inventaris dalam format tabel, pencarian barang termahal, perhitungan total nilai inventaris, pengecekan barang yang perlu restock (stok < 5), serta pembaruan stok secara fleksibel (tambah/kurangi). Dilengkapi validasi input agar tidak menerima nilai negatif atau format yang salah.

 - Project 4 : Game Minesweeper
Implementasi permainan Minesweeper di console dengan ukuran papan yang dapat dikustomisasi (4×4 hingga 10×10) dan jumlah bom yang bisa ditentukan sendiri. Pemain dapat membuka kotak, menandai kotak yang dicurigai mengandung bom, dan menyerah kapan saja. Fitur flood-fill otomatis membuka area kosong di sekitar kotak aman. Terdapat pencatatan waktu bermain dan nama pemain.

- Project 5 : Game Tebak Negara
Program kuis tebak nama negara yang menampilkan nama negara dengan sebagian hurufnya disembunyikan secara acak (diganti _). Pemain menebak nama negara yang dimaksud, mendapat skor jika benar, dan bisa memilih untuk terus bermain atau berhenti. Mencakup 70 negara dari berbagai benua dengan perbandingan jawaban yang tidak case-sensitive.
